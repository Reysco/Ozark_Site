# 📘 Ozark Consultoria — Landing Page Oficial

Este repositório contém o código-fonte da **landing page institucional da Ozark Consultoria**, desenvolvida em **React + Vite + TailwindCSS**.

O objetivo é apresentar os serviços da empresa, incluindo Governança Digital 360, Dados & Automação, Engajamento e Comunicação.  
A aplicação utiliza componentes visuais simples inspirados na biblioteca shadcn/ui e ícones do Lucide.

---

## 🚀 Tecnologias Utilizadas

- **React 18**
- **Vite 5**
- **TailwindCSS 3**
- **Lucide React Icons**
- **JavaScript (ES2023)**
- **Componentização funcional**

## 🔧 Como Rodar o Projeto Localmente

### 1. Clone o repositório

```bash
git clone https://github.com/SEU_USUARIO/ozark-site.git
```

### 2. Entre na pasta do projeto

```bash
cd ozark-site
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Rode o servidor de desenvolvimento

```bash
npm run dev
```

### 5. Acesse no navegador:

```
http://localhost:5173
```

---

## 🧱 Build para Produção

Para gerar os arquivos otimizados:

```bash
npm run build
```

Os arquivos finais ficarão em:

```
dist/
```

Para testar o build:

```bash
npm run preview
```
