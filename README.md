# Ozark Consultoria

Site institucional da **Ozark Consultoria**, desenvolvido para apresentar a empresa, seus serviços, produtos e fundadores, com foco em estratégia, dados, governança e execução.

## Visão Geral

A Ozark Consultoria atua ajudando empresas a crescer de forma estruturada, utilizando dados reais, processos bem definidos e decisões estratégicas fundamentadas.

O projeto é um site estático, responsivo, com foco em:
- Performance
- Clareza visual
- Experiência em desktop e mobile
- Identidade visual forte

## Tecnologias Utilizadas

- HTML5
- CSS3 (responsivo)
- JavaScript (menu mobile)
- Font Awesome (ícones)
- Google Fonts

## Funcionalidades

- Header fixo com menu responsivo
- Seção Hero com CTA
- Sobre a empresa (Missão, Visão e Valores)
- Produtos / Serviços
- Fundadores
- Galeria
- Formulário de contato
- Links externos (Instagram, WhatsApp, Chatbot)
- Download de políticas em PDF

O site é totalmente responsivo, adaptando-se corretamente para:
- Desktop
- Tablets
- Smartphones
